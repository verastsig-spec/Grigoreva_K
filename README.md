Currency Converter App

**Автор:** [Ксения Григорьева]

Описание
Приложение для мгновенной конвертации валют с использованием актуальных курсов и сохранением истории операций.

Как получить API-ключ
1. Зарегистрируйтесь на сайте [exchangerate-api.com](https://exchangerate-api.com).
2. Подтвердите почту.
3. Скопируйте "API Key" из личного кабинета.
4. Вставьте ключ в переменную `self.api_key` в коде программы.

Инструкция по запуску
1. Установите зависимости: `pip install customtkinter requests`
2. Запустите программу: `python main.py`

Тестирование
* **Положительный тест:** Введите 100 USD -> RUB. Ожидайте текущий курс.
* **Негативный тест:** Введите "-50" или "abc". Программа выдаст ошибку валидации.
